V-gDB Example API Scripts

To help users integrate V-gDB into their workflow, we provide a collection of example scripts demonstrating how to make API requests in different programming languages. Whether you're using Python, JavaScript, R, or another language, these samples offer a quick and easy way to interact with the database. Each script showcases essential API calls, from retrieving viral genome data to filtering results, so you can get started with minimal setup.

Two Ways to Use V-gDB: Hosted API or Local Installation

V-gDB offers two flexible ways to access viral genome data, depending on your needs:

  1. Hosted API Access

    • Use the predefined V-gDB API endpoint to directly connect to our curated viral genome dataset through simple API requests.
    • This option requires no setup — just make your requests to our server and start retrieving data right away.

  2. Local Installation with Custom Datasets

    • Install the V-gDB app locally to run the API on your own machine via localhost:8000.
    • This option allows you to use your own viral genome dataset, giving you full control over the data and environment for custom analyses.

Both methods offer the same functionality, with the freedom to choose between convenience or customization based on your project needs.
