V-gDB Python Tools

The V-gDB python tools are a series of scripts that allows you to access the database without using an API request. These tools only work on a local verson of V-gDB.

Install V-gDB

Follow the steps in the V-gDB installation guide to download and install the V-gDB project and connect your database.

Note: the python tools still run within a Django framework. The python tools allow you to access all the functions of the API without running a django server.

Link V-gDB to python scripts

Within your own python scripts, you need to link the V-gDB django project to access the V-gDB models and functions. 


            import os
            import sys
            import django

            # Example
            #TODO: you MUST add your absoulte local path to the base directory of V-gdb/backend
            project_root = os.path.abspath(os.path.join(os.path.dirname(__file__), '..', '..', 'backend'))
            sys.path.insert(0, project_root)


            sys.path.append(os.path.abspath(os.path.join(os.path.dirname(__file__), '..', '..')))

            os.environ.setdefault('DJANGO_SETTINGS_MODULE', 'backend.settings')

            # Initialize Django
            django.setup()

            # Now you can import the V-gDB helper classes
            from models.alignment_helper import Alignment

$ python
