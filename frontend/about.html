Viral Genome Toolkit

V-gTK allows you to create your own viral genome database by curating and structuring data curated from NCBI. It creates a viral database that can be used with our viral genome database (V-gDB).

Alternatively, if you prefer not to use the API, the database gives you the flexibility to work directly on your own. You can manage, query, and analyze the data independently, without relying on the API or the front-end display. This offers full control over your data and how you interact with it.

The V-gTK codebase can be downloaded from here: git hub link

The V-gTK installation instructions can found here: Link to README

Viral Genome Database

Once the database is created, you can use the V-gDB API to query this data, which is then displayed on the front end for easy access and visualization.

V-gDB relies on the database format outlined in core schema. If you wish to create your own database without the usage of V-gTK, for compatability with V-gDB API, please ensure the core tables and attributes outlined in the core schema are present.

The Viral Genome Web Resource

We have created a template GUI that links to the V-gDB API for basic visualization and analysis.

For a more adanced GUI, please refer to our pre-built web resources. These have advanced functions and styling.
