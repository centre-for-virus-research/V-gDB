Viral Genome Toolkit

V-gTK allows you to create your own viral genome database by curating and structuring data from NCBI.

You can manage, query, and analyze the data independently or interact with it through the API or frontend.

Download the V-gTK codebase: GitHub Link

Installation instructions: Link to README

Viral Genome Database

Once the database is created, use the V-gDB API to query data and visualize it through the frontend.

Refer to the core schema for database format requirements if creating your own database.

The Viral Genome Web Resource

We offer a GUI template linked to the V-gDB API for visualization and analysis. Advanced GUIs are also available.
